5-Stage Multi-Cycle Processor (February '15 - March '15)

Link to Project

  • Given an initial single-cycle processor, with only one stage of execution, my team first simulated a 3-stage pipelined processor.
  • Instruction forwarding was then implemented thus creating a multi-cycle processor, using pipeline registers with additional hazard detection.
  • The 3-stage pipelined processor was then fully optimized to a 5-stage pipelined processor with 4 pipeline registers, consisting of Fetch/Decode, Decode/Execute, Execute/Memory, Memory/Write, similar to the MIPS 5-stage processor.
  • When compared to the baseline single-cycle processor, the 5-stage pipelined processor resulted in a 34% speedup.
  • Technologies used include Altera Quartus and ModelSim, with System Verilog as the core programming language.
